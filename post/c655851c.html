毕业设计

研究内容：
革兰氏阳性菌分泌的非经典效应蛋白能从细菌细胞质进入宿主细胞，发挥毒力作用并使宿主致病。鉴于非经典效应蛋白的实验验证非常困难耗时，目前已存在多种基于机器学习的非经典效应蛋白预测工具。虽然预测新蛋白非常重要，但是对已有实验验证的非经典效应蛋白的收集，注释和分析，对于寻找此类蛋白的共同性质、功能具有重要意义。而且高质量的数据集也会极大地促进新的机器学习的开发。因此，本项目旨在开发一个存储和注释非经典效应蛋白、同时提供现有蛋白分析工具和未知蛋白预测工具的集成平台。
拟研究的方法：

  1. 采用基于JAVA的Web开发框架（包含Struts2和Hibernate3，前端开发为jQuery和Bootstrap）进行平台开发
  2. 采用ECharts等前端绘图或分析软件包或者软件，开发分泌效应蛋白的分析功能
  3. 采用隐马尔科夫链模型开发高性能的预测器并集成本实验室最新开发的基于机器学习的预测工具进行蛋白预测

存在的问题

  • 在毕设研究的题目中集成的预测平台为bastion6，bastion6用于准确预测VI型分泌效应物的生物信息学方法许多革兰氏阴性细菌使用VI型分泌系统（T6SS）将效应蛋白输出到相邻的靶细胞中。
    http://bastion6.erc.monash.edu/index.jsp ，是否需要替换模型

  • 关于使用hmmer的合理性

    在使用hmmer过程中需要到了Clustal去生成多序列比对信息，多序列比对反映的是蛋白质间的亲缘关系。但是对于革兰氏阳性菌分泌的非经典蛋白，在使用Clustal生成的多序列比对信息，得到的序列比对成功密度很小，而hmmer提供的globins蛋白质序列生成的对序列比对结果序列比对成功的密度很大，创建出来的hmmer模型效果很好，对于原数据集中分出的测试集测试都成功有结果出现。

springboot

今天帮一个同学解决问题中解决了我一个一直以来的心病，就是springboot中关于静态资源加载问题。
在使用前端模板thymeleaf中加载静态资源标签的时候以前一直都是用

但是在上个学期末还有刚才一直报错404，网上关于静态资源映射方法基本都是修改配置文件。然后今天碰到了这个问题，换了个姿势重新谷歌了一下，然后就茅厕顿开了，https://blog.csdn.net/zhoucheng05_13/article/details/78197766

在SpringBoot中，默认配置的/映射到/static。
静态资源映射还有一个配置选项，为了简单这里用.properties方式书写：
spring.mvc.static-path-pattern=/ # Path pattern used for static
resources.
这个配置会影响默认的/，例如修改为/static/后，只能映射如/static/js/sample.js这样的请求（修改前是/js/sample.js）。这个配置只能写一个值，不像大多数可以配置多个用逗号隔开的。

所以以前的问题我不知道是怎么解决的。。。也不知道原因，看了这个博文就知道自己的bug一直出在/static/这里。

SVG

指可伸缩矢量图形 (Scalable Vector Graphics)
用来定义用于网络的基于矢量的图形
使用 XML 格式定义图形
图像在放大或改变尺寸的情况下其图形质量不会有所损失
是万维网联盟的标准
与诸如 DOM 和 XSL 之类的 W3C 标准是一个整体

实例http://www.w3school.com.cn/svg/svg_examples.asp
